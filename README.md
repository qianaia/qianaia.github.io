# 📚 [青春向：《梧桐道的第三片落叶》](https://qianaia.github.io/)
九月的风裹着梧桐叶的气息，吹过明德大学的主干道，林知夏抱着厚厚的专业书，在第三片落叶飘下来的时候，撞进了陈桉的怀里。

“对不起对不起！”她慌忙道歉，指尖触到他白色衬衫的袖口，像触到一片刚融化的雪。 陈桉弯腰捡起她掉落的钢笔，笔帽上刻着小小的“夏”字。“你的笔，”他笑起来的时候，左边有个浅浅的梨涡，“下次走路看路，梧桐叶不会撞疼你，但我会。”

后来知夏发现，陈桉总是出现在她的世界里：图书馆靠窗的邻座，高数课最后一排的斜前方，甚至是她常去的食堂窗口。他会在她被难题困住时，递来一张写满解题步骤的草稿纸；会在降温的傍晚，把自己的外套披在她肩上；会在她熬夜准备辩论赛时，悄悄放在桌角一杯热可可。

毕业那天，梧桐道的叶子落了一地，陈桉把一个盒子递给她，里面是三百六十五片梧桐叶，每一片上都写着一句话。最后一片叶子上，他用钢笔写着：“第三片落叶的相遇，是我蓄谋已久的靠近。” 知夏抬头时，看到陈桉眼里的星光，比梧桐道的路灯还要明亮。那天他们牵起手，风穿过梧桐叶的缝隙，像是在说：原来最好的爱情，是刚好在青春里，遇到那个愿意和你一起捡落叶的人。
